# 🤖 IntelliChat - AI-Powered Chatbot with DeepSeek Integration

[![Python Version](https://img.shields.io/badge/Python-3.8%2B-blue)](https://python.org)
[![Flask](https://img.shields.io/badge/Flask-2.0%2B-green)](https://flask.palletsprojects.com)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A modern web-based chatbot application powered by DeepSeek's AI API, featuring dark mode, chat history, and interactive UI components.


## ✨ Features

- 🧠 Real-time AI conversations using DeepSeek's powerful language model
- 🌓 Smart theme switching (Light/Dark mode)
- 📜 Interactive chat history sidebar
- 💬 Responsive and modern UI design
- 🔒 Session-based conversation storage
- 🚀 Easy API integration
- 📱 Mobile-friendly interface
